Desarrolla un sitio web corporativo moderno para “Marquez & Aguirre Ingeniería (M&A)”, una empresa en Chile que presta servicios de ingeniería estructural, sanitaria, eléctrica (TE1), IMIV, permisos de edificación, regularizaciones, obras menores y apoyo técnico integral.
Objetivo principal: generar confianza, mostrar servicios/proyectos y captar contactos/cotizaciones. No se requieren pagos ni e-commerce.

BRANDING (usar logo adjunto)

Usar el logo de M&A en navbar, loading y footer.

Paleta basada en el logo:

Fondo: blanco #FFFFFF

Texto principal: negro #0A0A0A

Acento principal (azul M&A): #2F2F92

Neutros (para bordes/lineas/sombras suaves): #D1D2E3, #8786A1

Estilo: sobrio, técnico, confiable, tipo “estudio/consultora de ingeniería”.

Tipografía: sans-serif moderna y geométrica para títulos (estilo Outfit/Inter), sans-serif legible para párrafos. Opcional: monospace solo para micro-detalles técnicos (muy sutil).

ESTRUCTURA DEL SITIO

Construir como Single Page con secciones (y opcional páginas internas para “Servicios” y “Proyectos”).
Navegación fija superior con enlaces: Inicio / Servicios / Proyectos / Proceso / Sobre nosotros / Contacto.

Frase breve de valor:

“Diseñamos, calculamos y tramitamos con foco en seguridad, normativa y plazos.”

Botones CTA:

“Estructural” · “Sanitaria” · “Eléctrica (TE1)” · “IMIV” · “Permisos y regularizaciones”

Animación sutil: fade/slide al cargar + indicador de scroll.

Detalles · cubicaciones básicas · soporte técnico y coordinación

Cada servicio con botón: “Consultar” que haga scroll a Contacto y preseleccione el tipo de proyecto.

Cada proyecto incluye:

Nombre del proyecto (placeholder)

Disciplina: Estructural / Sanitaria / Eléctrica / IMIV / Permisos

Ubicación: (Ciudad, Chile) (placeholder)

Tags: “Memoria de cálculo”, “Planos”, “Tramitación”, etc.

Hover: zoom suave + overlay “Ver detalle”.

Detalle de proyecto (modal o página):

Resumen con: tipo, año (placeholder), alcance, entregables, herramientas (genéricas), desafíos/solución.

Galería de imágenes.

CTA: “Quiero un proyecto similar”.

CARACTERÍSTICAS TÉCNICAS Y UX

Diseño responsive (mobile primero).

Smooth scroll + animaciones sutiles (Framer Motion o equivalente).

Performance: lazy loading de imágenes.

Accesibilidad: contraste correcto, focus visible, labels.

SEO básico: title, description, Open Graph.

Modo oscuro: opcional (mantener identidad, azul M&A como acento).

CONTENIDO

Todo el contenido en español (Chile), tono formal y claro.

Evitar claims no verificables (certificaciones, números reales). Usar placeholders editables.

Dejar una estructura fácil de editar (por ejemplo data/services.json y data/projects.json).
